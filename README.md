# SETK4133-ENERGY-PLANNING

This course provides students with the ability to use optimisation software such as Generalised Algebraic Modeling Systems (GAMS), AIMMS as a tool for efficient and holistic energy planning.   Students are expected to develop the strategic energy planning for micro and macro level includes industry, regional and country-wide. Emphasis will be placed on the formulation of mathematical model, interpret meaningful scenario analysis in engineering, science and business. A series of hands-on exercises will be conducted in class to enhance the understanding.
